# Calm Tissue website redesign

Static, production-ready corporate site. No build step or runtime dependencies are required.

The navigation and footer use the supplied Calm Tissue logo artwork without alteration. Primary brand colours are Calm Blue `#0033A0` and Calm Sky `#95D9EA`.

## Company journey

Calm Tissue was founded in South Africa by a Motswana in 2024, traded through channels including SPAR, Makro Online, Uber Eats and other chain stores, and opened its Botswana factory in Selibe Phikwe in 2025. Retailer references describe historical sales channels and must not be presented as endorsements.

## Preview

Serve the `site` directory with any static server, for example `npx serve site`, then open the local URL.

## Deploy

Upload the contents of `site/` to the web root, or configure GitHub Pages/Cloudflare Pages with `site` as the publish directory.
